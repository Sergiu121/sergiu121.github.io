Exerciții de prelucrare a datelor
=================================

Tutorial: Analiza evoluției cursului valutar USD-EUR: https://data.humdata.org/dataset/ecb-fx-rates

Avem un catalog în format .xls pe care realizăm diferite prelucrări, precum cele din cursul de automatizarea sarcinilor. Acest lucru se face în Google Docs sau LibreOffice, în format interactiv.
Apoi pentru cataloage din mai mulți ani, care necesită folosirea scriptingului.

Să extragă ultima versiune de fișier CSV de GitHub ranking și să facă un script care afișează top 10 repository-uri pentru un limbaj primit ca argument. Le dăm scheletul în Bash și Python și folosesc ce doresc ei.

Prelucrare pe istoricul Bitcoin (https://tokendatabase.com/): când a fost topul Bitcoin, în ce zile a crescut cel mai mult.
**Bonus**: Cei care pot și vor, să facă un grafic pe baza prelucrării.

Să intre pe datahub.io (https://datahub.io/) să caute un tip de informație care îi interesează, să facă o prelucrare care îi interesează.

Avem informații în format .JSON extrase folosind testssl.sh. Mai multe site-uri interogate legat de capacitățile TLS/SSL. Apoi făcut prelucrări pe toate aceste informații format JSON.

Avem fișiere în format CSV și format JSON legate de apeluri de sistem (dintr-un proiect Unikraft). Dorim să extragem informații din aceste fișiere.

**Bonus**: Ceva cu prelucrare interactivă în Vim (substituții cu regex, recording)
