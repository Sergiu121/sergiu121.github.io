Vizualizarea configurației sistemului
=====================================

**tutorial**: pe un sistem Linux este instalată o distribuție
cat /etc/os-release, /etc/lsb-release, lsb_release -a, https://whatsmyos.com/

**tutorial**: există comenzi / fișiere generice pentru informații
cat /proc/version, uname -a, cat /etc/issue

**tutorial**: există comenzi / fișiere particulare
http://linuxmafia.com/faq/Admin/release-files.html

**breviar**: în general, o aplicație poate rula pe orice distribuție

**breviar**: dar instalarea aplicației va folosi alte utilitare, la fel anumite configurări în sistem
