.. _app_install_update:

Actualizarea aplicațiilor
=========================

**breviar**: aplicațiile sunt actualizate de dezvoltatori (mai multe funcționalități, securitate, bug fixes); versiunea dintr-o distribuție primește doar actualizări de securitate și de tip bug fix, nu funcționalitate

**tutorial**: actualizarea se poate face pentru o aplicație
sudo apt install <app>

**tutoarial**: sau pentru întreg sistemul
sudo apt update
sudo apt upgrade
sudo apt dist-upgrade

**breviar**: în general sistemul notifică utilizatorul de actualizări
