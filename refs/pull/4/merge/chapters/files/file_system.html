Sistemul de fișiere¶

Find¶

Pentru a căuta un fișier într-o ierarhie de directoare, putem folosi utilitarul find. Pentru a învăța mai multe despre acest utilitar, putem inspecta manualul man find. Acesta nu scalează pentru situațiile când avem multe fișiere și dorim un răspuns rapid.

Locate¶

Utilitarul este simplu de folosit: locate nume_fisier. Utilitarul funcționează în 2 pași: 1) Crearea și actualizarea unei baze de date folosind comanda updatedb; 2) Căutarea folosind locate.

Partea negativă este că la fiecare adăugare/ștergere de fișier, este necesară actualizarea baze de date. Partea pozitivă este performanța acestuia.

Exercițiu: Căutați utilitarul pwd în sistemul de fișiere folosind find și locate.

PATH¶

După cum am menționat într-un capitol anterior, PATH este o variabilă de mediu ce se află în contextul executării în linia de comandă. De fiecare dată când apelăm un program prima oară se verifică dacă în căile din PATH se află acest program.

Exercițiu: Afișează valoarea PATH a sistemului tău.

Which¶

Acest utilitar identifică calea programelor din sistem. Pentru mai mute detalii verificați manualul man which.
