Editor fișier¶

1) Editor grafic¶

Editoare grafice: sublime, gedit,

Exercițiu: * Deschideți fișierul Hulu/Favorite creat anterior cu editorul sublime; * Scrieți textul U50 RU|3Z; * Duplicati textul scris anterior (copiat/lipit; * Indentați (adăugați 4 spații la început) fiecare rând; * Adăugați caracterul # la început de rând pentru linia 2;
