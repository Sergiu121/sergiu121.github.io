3. Configurarea aplicațiilor¶

breviar: o aplicație poate rula diferit conform configurației

breviar: configurația e stabilită prin fișier de configurare sau prin opțiuni de pornire

breviar: formele de folosire a ps -e -f sunt configurare de pornire

exercițiu: putem porni o aplicație cu opțiuni cu Alt+F2 TODO exemplu

tutorial: fișierul de configurare se poate edita din interfața grafică exemplu Sublime

tutorial: fișierul de configurare se poate edita cu o comandă exemplu git config

tutorial: fișierul de configurare se poate edita cu editorul exemplu ~/.vimrc

exercițiu: de editare VirtualBox din GUI, văzut modificarea în .vbox
