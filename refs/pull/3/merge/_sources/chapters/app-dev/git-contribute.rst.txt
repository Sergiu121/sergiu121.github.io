Contribuirea la un alt repository
=================================

1) Clonarea repository-ului
---------------------------

Include crearea, validarea lui, și GitHub Student Pack.

2) Crearea unui branch de lucru
-------------------------------

``git checkout -b <my-branch>``

3) Crearea unui commit cu noile schimbări în repository
-------------------------------------------------------

``git add`` + ``git commit -m`` + ``git push``

4) Crearea unui pull request (PR) pe GitHub
-------------------------------------------

cum se crează, cum adaugi recenzori, cum poți face recenzie la PR

5) Rezolvarea conflictelor și comasarea în branch-ul master
-----------------------------------------------------------

din interfața grafică GitHub

6) Ștergerea branch-ului în urma comasării în branch-ul master
--------------------------------------------------------------

ștergerea locală și la distanță
